# AI短剧协作台 - UI设计规范

## 一、设计理念

### 核心定位
电影工业级创作协作平台，而非传统管理后台。

### 设计关键词
- **电影质感** - 深色主题 + 金色光效，营造影院级沉浸体验
- **高级感** - 毛玻璃面板 + 渐变光影 + 精致排版
- **流畅交互** - 优雅动画 + 微反馈 + 状态可视化

---

## 二、色彩系统

### 主色调
| 用途 | 颜色 | 色值 | 说明 |
|------|------|------|------|
| 背景主色 | 深黑 | `#0a0a0f` | 纯黑底色，营造沉浸感 |
| 背景辅色 | 暗黑 | `#12121a` | 用于次级背景 |
| 强调色 | 金色 | `#ffd700` | 主品牌色，象征电影工业 |
| 渐变金 | 浅金→深金 | `#ffd700 → #ffb700` | 按钮、Logo、高光 |

### 功能色
| 类型 | 颜色 | 色值 | 用途 |
|------|------|------|------|
| 成功 | 翠绿 | `#10b981` | 完成状态、通过按钮 |
| 警告 | 琥珀 | `#f59e0b` | 进行中状态、待处理 |
| 危险 | 赤红 | `#ef4444` | 驳回、错误提示 |
| 信息 | 宝蓝 | `#3b82f6` | 信息提示、链接 |

### 文字色阶
| 层级 | 颜色 | 色值 | 用途 |
|------|------|------|------|
| 主标题 | 纯白 | `#ffffff` | 页面标题、重要信息 |
| 正文 | 浅灰 | `#e5e7eb` | 常规文字内容 |
| 次要 | 中灰 | `#9ca3af` | 辅助说明 |
| 弱化 | 深灰 | `#6b7280` | 时间戳、标签 |

## 三、排版系统

### 字体规范
- **字体族**: -apple-system, BlinkMacSystemFont, "Segoe UI", "PingFang SC", "Microsoft YaHei"
- **字重层级**:
  - Black (900) - 页面主标题、重要数据
  - Bold (700) - 卡片标题、按钮文字
  - Semibold (600) - 小标题、标签
  - Medium (500) - 导航、链接
  - Regular (400) - 正文

### 字号层级
| 用途 | 字号 | 字重 | 示例 |
|------|------|------|------|
| 页面大标题 | 28px | 700 | "项目详情" |
| 卡片标题 | 18px | 700 | 项目名、任务名 |
| 数据展示 | 36-48px | 900 | 项目数量、待办数 |
| 正文 | 14px | 400 | 描述文字 |
| 小字 | 12px | 400 | 标签、时间戳 |

### 间距系统
- **基础单位**: 4px
- **常用间距**: 8px, 12px, 16px, 24px, 32px, 40px, 48px
- **卡片内边距**: 24px
- **页面内容区边距**: 32px 40px

## 四、组件规范

### 1. 卡片 (Card)
**视觉特征**:
- 毛玻璃背景：半透明白色 + backdrop-filter blur(20px)
- 边框：1px 半透明金色 `rgba(255,215,0,0.15)`
- 圆角：16px
- 顶部高光：1px 金色渐变线
- 悬停效果：边框变亮 + 阴影增强 + 微上浮 2px

**使用场景**: 任务卡片、项目卡片、表单容器、信息面板

### 2. 按钮 (Button)

#### 主按钮 (Primary)
- **背景**: 金色渐变 `linear-gradient(135deg, #ffd700, #ffb700)`
- **文字**: 深黑 `#0a0a0f`
- **阴影**: `0 4px 15px rgba(255,215,0,0.3)`
- **悬停**: 渐变变亮 + 阴影增强 + 上浮 1px
- **用途**: 核心操作（保存、提交、创建）

#### 次要按钮 (Secondary)
- **背景**: 半透明白色 `rgba(255,255,255,0.08)`
- **边框**: 1px `rgba(255,255,255,0.15)`
- **文字**: 浅灰 `#e5e7eb`
- **悬停**: 背景变亮 + 边框变金色
- **用途**: 次要操作（返回、取消、刷新）

#### 特殊按钮
- **文生图**: 金色渐变（同主按钮）
- **图生视频**: 紫色渐变 `#a855f7 → #7c3aed`
- **通过**: 绿色渐变 `#10b981 → #059669`
- **驳回**: 红色渐变 `#ef4444 → #b91c1c`

### 3. 输入框 (Input/Textarea)
- **背景**: 半透明白色 `rgba(255,255,255,0.06)`
- **边框**: 1px `rgba(255,255,255,0.12)`
- **圆角**: 10px
- **内边距**: 12px 16px
- **聚焦状态**: 边框变金色 + 发光阴影 `rgba(255,215,0,0.15)`

### 4. 状态标签 (Badge)
- **形状**: 圆角胶囊
- **内边距**: 6px 14px
- **字号**: 12px，字重 600
- **颜色方案**:
  - 金色标签：金底 + 金边 + 金字
  - 蓝色标签：蓝底 + 蓝边 + 蓝字
  - 绿色标签：绿底 + 绿边 + 绿字
  - 琥珀标签：琥珀底 + 琥珀边 + 琥珀字
  - 红色标签：红底 + 红边 + 红字

### 5. 顶部导航 (Top Navigation)
**布局结构**:
- 高度：64px
- 背景：深黑渐变 + 毛玻璃
- 底边框：1px 金色渐变高光
- 阴影：底部投影

**内容区域**:
- 左侧：品牌 Logo（金色渐变方块）+ 平台名称
- 中部：导航链接（当前页高亮金色）
- 右侧：用户头像（金色圆形）+ 用户名 + 退出按钮

### 6. 页面标题栏 (Page Header)
- 内边距：32px 40px
- 背景：半透明白色渐变
- 底部：金色渐变高光
- 标题：28px 白色粗体
- 副标题：14px 灰色

### 7. 步骤条 (Step Bar)
**节点状态**:
- **激活态**: 金色渐变圆形 + 发光效果
- **完成态**: 绿色渐变圆形 + 对勾图标
- **待处理**: 半透明白色圆形 + 灰色边框

**连接线**: 渐变线条，带金色光效

### 8. 模态框 (Modal)
- **遮罩**: 80% 黑色 + 毛玻璃模糊
- **内容区**: 白色卡片 + 大圆角
- **按钮**: 左侧取消（灰色），右侧确认（功能色）

### 9. 通知提示 (Toast)
- **位置**: 右上角固定
- **样式**: 毛玻璃背景 + 圆角 + 阴影
- **颜色**: 根据类型（成功绿、警告黄、错误红、信息蓝）
- **动画**: 滑入 + 淡出

## 五、页面布局

### 1. 登录页
- **背景**: 深色渐变 + 径向光效（金色光斑）
- **卡片**: 居中毛玻璃面板
- **Logo**: 大号金色渐变方块 + 脉冲光效
- **标题**: 40px 白色超粗体
- **按钮**: 全宽金色渐变

### 2. 工作台（首页）
- **欢迎区**: 大标题 + 金色数字统计卡片
- **任务区**: 双列卡片网格
- **项目区**: 三列卡片网格
- **操作**: 顶部刷新按钮 + 创建项目链接

### 3. 项目详情
- **顶部**: 项目标题（36px）+ 步骤条
- **中部**: 三栏信息卡片（进度、阻塞、资产）
- **底部**: 时间线（竖向滚动）

### 4. 剧本工位
- **布局**: 居中窄栏（max-width: 4xl）
- **表单**: 全宽输入框 + 大文本域
- **操作**: 底部按钮组（保存 + 提交）

### 5. 分镜工位
- **三栏布局**:
  - 左侧（25%）：分镜卡片列表（可滚动）
  - 中部（42%）：大画面预览区（16:9 比例）
  - 右侧（33%）：编辑表单
- **操作**: 顶部生成按钮 + 底部保存/提交

### 6. 生成队列
- **布局**: 双列卡片网格
- **每张卡片**: 场景号 + 状态标签 + 描述 + 操作按钮
- **操作**: 生成图片（金色）+ 生成视频（紫色）

### 7. 审核会话
- **左侧（67%）**: 分镜预览网格（双列）
- **右侧（33%）**: 历史审核记录（滚动）
- **操作**: 顶部通过（绿色）+ 驳回（红色）按钮

### 8. 导出中心
- **顶部**: 状态信息 + 操作按钮
- **中部**: 资产清单列表（链接列表）

## 六、动效规范

### 页面级动画
- **淡入上移**: 0.5s，cubic-bezier(0.4, 0, 0.2, 1)
- **应用场景**: 所有页面加载

### 组件级动画
- **卡片悬停**: 0.4s，上浮 2px + 阴影增强 + 边框变亮
- **按钮悬停**: 0.3s，渐变色变化 + 阴影增强 + 上浮 1px
- **输入框聚焦**: 0.3s，边框变色 + 发光阴影
- **导航链接**: 0.3s，背景色变化 + 文字变色

### 特殊动效
- **Logo 脉冲光效**: 3s 循环，金色阴影呼吸
- **Toast 通知**: 滑入（右→左）+ 淡出
- **模态框**: 遮罩渐显 + 内容区缩放进入

## 七、响应式策略

### 断点
- **桌面端**: ≥1280px（主设计目标）
- **平板端**: 768px - 1279px（双列布局）
- **移动端**: <768px（单列布局）

### 自适应规则
- 导航栏：平板/移动端隐藏部分链接
- 网格布局：从多列自动降为单列
- 卡片间距：移动端减小内边距

## 八、设计原则

### 1. 一致性
- 所有卡片使用相同样式
- 按钮颜色遵循功能语义
- 间距统一使用 4px 倍数

### 2. 层次感
- 通过字号、字重、颜色建立信息层级
- 重要操作用金色，次要操作用灰色
- 数据用大字突出，说明用小字弱化

### 3. 沉浸感
- 深色背景减少视觉干扰
- 毛玻璃效果增加深度感
- 金色光效营造电影氛围

### 4. 反馈性
- 所有交互元素都有悬停状态
- 操作有即时视觉反馈（按钮、卡片、输入框）
- 状态变化有动画过渡

## 九、视觉元素库

### 渐变方案
```
金色主渐变: linear-gradient(135deg, #ffd700, #ffb700)
紫色渐变: linear-gradient(135deg, #a855f7, #7c3aed)
绿色渐变: linear-gradient(135deg, #10b981, #059669)
红色渐变: linear-gradient(135deg, #ef4444, #b91c1c)
背景渐变: linear-gradient(180deg, rgba(10,10,15,0.98), rgba(10,10,15,0.95))
```

### 阴影方案
```
卡片默认: 无（靠边框区分）
卡片悬停: 0 8px 32px rgba(0,0,0,0.4) + 0 0 40px rgba(255,215,0,0.1)
按钮默认: 0 4px 15px rgba(255,215,0,0.3)
按钮悬停: 0 6px 20px rgba(255,215,0,0.5)
导航底部: 0 4px 20px rgba(0,0,0,0.5)
```

### 光效方案
```
金色边框高光: linear-gradient(90deg, transparent, rgba(255,215,0,0.5), transparent)
卡片顶部光: linear-gradient(90deg, transparent, rgba(255,215,0,0.4), transparent)
背景光斑: radial-gradient(circle, rgba(255,215,0,0.08), transparent 50%)
```

## 十、品牌标识

### Logo 设计
- **图形**: 20x20px 圆角方块
- **背景**: 金色渐变
- **文字**: "AI" 深黑超粗体
- **效果**: 脉冲光效（金色阴影呼吸）

### 品牌色应用
- 主操作按钮
- 激活态导航链接
- 重要数据展示
- 步骤条激活节点
- Logo 和用户头像

### 品牌调性
- **专业**: 电影工业级质感
- **高级**: 金色 + 毛玻璃 + 精致细节
- **沉浸**: 深色背景 + 光效
- **现代**: 圆角 + 渐变 + 流畅动画
