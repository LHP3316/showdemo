# 前端 Static 页面与 PRD 差距清单

本文档用于对照 `AI短剧协作平台 - 产品需求文档.md` 与当前 `frontend/static` 页面现状，整理尚未完成或仅部分完成的内容，便于按阶段逐步处理。

## 目标

- 以 PRD 为主，保留当前 `static` 页面视觉风格
- 优先补齐 MVP 主流程
- 尽量先做“低改动、高收益”的功能，再做结构性改造

## 当前已完成的关键项

- 登录 / 注册基础流程可用
- 注册已支持角色选择：`director` / `staff`
- 注册已支持 `display_name`
- 工作台已支持按角色切换为导演 / 工作人员两种视图
- 工作人员只能看到分配给自己的项目
- 导演可以创建项目
- 导演可以给项目分配工作人员
- 分镜工位已补齐部分核心字段：
  - 角色
  - 场景描述
  - 台词
  - 情绪
  - Prompt
- 生成队列页已接真实数据
- 审核页已可提交通过 / 驳回并查看历史
